Selenium é um framework que permite você automatizar atividades no seu navegador, como fazer login, "confirmar que você é um humano" e extrair informações de páginas dinâmicas, onde o simples uso do Scrapy não seria possível.

Nesta aula, iremos fazer o login no LinkedIn, fazer uma busca no Google por "Cientistas de Dados" na cidade de São José dos Campos e extrair as suas informações para um arquivo csv.

Deixo também como exercício, você se conectar a alguns desses contatos no LinkedIn (usando o Selenium).

Para instalar a biblioteca, basta executar o comando pip install selenium
